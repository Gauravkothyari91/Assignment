5. What is emmet? List some of the advantages emmet offers?

EMMET

Emmet is a plugin for text editors that allows for faster and more efficient HTML and CSS coding. It allows developers to write HTML and CSS code using shorthand notation, which is then expanded into full code by the plugin.

Some of the advantages of emmets are-
Saves a lot of time.
Increase productivity.
Efficient Programming.
Allows to type shortcuts that are then expanded into full pieces of code after auto suggestive letter pops up.
